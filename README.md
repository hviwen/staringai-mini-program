# 星搭小星 - ChatGPT 聊天小程序

对接 ChatGPT 的 OpenAI 接口，在国内实现与 ChatGPT 实时聊天，包含会话界面和聊天界面，一键分享聊天记录到微信聊天和朋友圈

![聊天](./images/3.png)

### 快速开始

1. 复制 `src/config.example.ts`

2. 填充小程序的 appId, 
```ts
export const appId = ''
```

3. 安装依赖并启动
```shell
yarn
yarn run start
```

打开微信开发者工具，引入
